# India Election Results Analysis (Power BI Project)

## Project Overview

This project presents an interactive Power BI dashboard analyzing Indian election results. The objective is to explore party performance, vote share distribution, and regional trends to derive meaningful insights from electoral data.

## Tools and Technologies

* Power BI
* Data Visualization
* Power Query (Data Cleaning and Transformation)
* DAX (Data Analysis Expressions)
* Excel (Data Source)

## Dataset Description

The dataset contains election results data including:

* Party-wise performance
* Vote share distribution
* Constituency-level results
* Regional breakdown of votes

## Dashboard Features

* Party-wise seat distribution
* Vote share analysis across parties
* Region/state-wise election results
* Comparison of party performance across regions
* Interactive filters for detailed exploration

## Key Insights

* Identified leading parties based on seat count and vote share
* Observed regional dominance patterns of political parties
* Highlighted variations in voter preferences across states
* Compared overall performance trends across different regions
